# Project Change Request

## Proposed Change

Add an embedded Google Map to the homepage that displays the company's hometown. The map will allow visitors to view the business location directly from the website and easily access directions.

## Reasons for the Change

The stakeholder requested this feature to make the website more informative and user-friendly. Including a map helps visitors quickly identify where the business is located and provides an easy way to find directions without leaving the website.

## How This Change Will Add Value

Adding a Google Map improves the user experience by making the website more useful for potential customers. It increases the professionalism of the website and helps build trust by showing a physical location for the business.

## How This Change Will Affect the Constraints

### Time

The change would require a small amount of additional development time to obtain the Google Maps embed code, insert it into the webpage, and test that it displays correctly.

### Quality

This change would improve the overall quality of the website by making it more informative and easier to use.

### Budget

Because the project is being completed using free tools and an embedded Google Map, there would be little to no additional cost associated with implementing this feature.
