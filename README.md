# Final Project Website

## Project Overview

This project was created as the final project for CMPA 3301 at Texas Tech University. The purpose of this project is to demonstrate the skills learned throughout the course, including project planning, HTML, CSS, GitHub, GitHub Pages, and technical documentation.

The website consists of two pages and serves as a professional portfolio website that highlights my background, business information, and educational content.

## Project Features

- Two-page website (Home and About)
- Consistent navigation between pages
- External CSS stylesheet
- Professional layout and design
- Embedded Google Map
- Information about diabetes and hypertension
- Hosted using GitHub Pages
